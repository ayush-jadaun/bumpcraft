# Bumpcraft Examples

Production-ready examples showing how to integrate Bumpcraft into real workflows.

| Example | Description |
|---------|-------------|
| [github-actions/](github-actions/) | Automated releases on every push to main via GitHub Actions |
| [gitlab-ci/](gitlab-ci/) | Automated releases in GitLab CI/CD pipelines |
| [programmatic/](programmatic/) | Use Bumpcraft as a library in Node.js scripts |
| [monorepo-groups/](monorepo-groups/) | Batch releases across multiple PRs using release groups |
| [docker-api/](docker-api/) | Run Bumpcraft as an API server behind nginx with HTTPS |
